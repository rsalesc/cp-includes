### Guidelines

- There will be no `class` usage in this library, only `struct` usages.
- Everything will be wrapped in `namespace lib`.
- Every file will include _only_ `bits/stdc++.h`.
- Only `cpp` files allowed.

### Template

```cpp
// #include "<FILENAME>"
#include <bits/stdc++.h>

namespace lib {
  using namespace std;
}  // namespace lib

// end of #include "<FILENAME>"
```
